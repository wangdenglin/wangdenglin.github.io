js总结_Dom基本操作

JS dom基本操作
获取节点
document.getElementById(元素ID);
document.getElementsByName(元素name);
document.getElementsByTagName(元素标签);
节点指针
firstChild:获取元素的首个子节点;
lastChild:取元素的最后一个子节点;
childNodes:获取元素的子节点列表;
parentNode:获取已知节点的父节点;
previousSibling:获取已知元素的前一个节点(同级);
nextSibling:获取已知元素的后一个节点(同级);


节点操作
1.创建节点
createElement 创建元素节点,创建p标签document.createElement("p");
createAttribute 创建元素属性,document.createAttribute(元素属性);
createTextNode 创建文本节点,document.createTextNode(文本内容);
2.插入节点
appendChild 向节点的子节点列表的末尾添加新的子节点
insertBefore 在已知子节点前插入一个新的子节点
3.替换节点
replaceChild 将某个子节点替换成另一个,replaceChild('要插入的新元素','将被替换的老元素')
4.复制节点
cloneNode创建指定节点的副本，需要被复制的节点.cloneNode(true/false)
5.删除节点
removeChild 删除指定节点， removeChild(要删除的节点)
属性操作
getAttribute获取元素属性;
setAttribute设置元素属性;
removeAttribute移除元素属性;
文本操作
insertData(offset,String) 从offset的指定位置插入string;
appendData(String) 将string插入文本节点的末尾处;
deleteData(offset,count) 从offset起删除count个字符;
spiltData(offset) 从offset处将文本节点分为两个节点;
substring(offset,count) 返回由offset起的count个节点;

js总结_函数基础

JS函数基础
1.调用方法
直接调用：函数名(实参列表)
在链接中调用：<a href="javascript:函数名()">啦啦啦啦</a>
在事件中调用：事件类型=”函数名()”;
递归调用：在函数内部调用函数自身，function aaa(){alert('1111');aaa()}


2.方法
apply 将函数作为对象的方法来调用，将参数以数组形式传递给该方法;
call 将函数作为对象的方法来调用，将指定参数传递给该方法;
toString 返回函数的字符串表示
apply和call的区别是传入参数的形式不一样，其作用并无本质差别,都是改变this的指向。第一个是改变改变this的指向，其后就是传入的参数
如 func.call(func1,var1,var2,var3)对应的apply写法为：func.apply(func1,[var1,var2,var3]);

下例分别弹出了func和var。先调用func函数，用this.a=”func”替换myfunc中的this.a ; 然后将”var”传递给方法myfunc的参数x可见分别弹出了func和var。
对于apply和call两者在作用上是相同的，但两者在参数上有区别的。
第一个参数意义都一样。第二个参数：apply传入的是一个参数数组，也就是将多个参数组合成为一个数组传入，而call则作为call的参数传入（从第二个参数开始）。
如 func.call(func1,var1,var2,var3) 对应的apply写法为：func.apply(func1,[var1,var2,var3])，同时使用apply的好处是可以直接将当前函数的arguments对象作为apply的第二个参数传入。

js总结_数组

JS数组
1.创建数组

空数组：var Obj=new Array() 或var Obj=[];
指定长度的数组：var Obj=new Array(Size);
指定元素的数组：var Obj=new Array(元素1,元素2...)
单维数组 ：var Obj=[元素a,元素b,元素c...];
多维数组 ：var Obj=[[a],[b],[c]]
2.删除数组
delete 数组名加下标
3.遍历数组
for…in…
var example=[1,2,3,4];
for(a in example){ console.log('a在数组里'); }
4.数组的添加与删除
push() 在数组末尾添加数组
unshift() 在数组头部添加元素
concat() 合并两个数组
pop() 删除并返回数值的最后一个元素
shift() 删除并返回数组的第一个元素
splice() 可以在指定位置删除插入或替换
slice() 从已有数组中选取部分元素组成数组，如果是正值从前外后，如果是负值从后往前
reverse() 颠倒数组元素的顺序
sort() 排序


toString() 转化成字符串并返回 var aaa=[1,2,3,4] console.log(aaa.toString()) 输出1,2,3,4
join() 用指定分隔符分割数组并转化成字符串 var aaa=[1,2,3,4] console.log(aaa.join('-')) 输出1-2-3-4
indexOf() 从数组起始位置开始查找，如果有就返回>=0
every() 如果该函数的每项都为true,则返回true
some() 如果该函数的有一项都为true,则返回true
map() 返回每次函数调用的结果数组
forEach():没有返回值，只是针对每个元素调用func

你可以用以下命令查看所有已经安装的插件

cordova plugin ls

使用以下命令删除插件：

cordova plugin rm org.apache.cordova.console

或者通过地址来添加插件：

cordova plugin add https://github.com/apache/cordova-plugin-console.git

帮助：

cordova help

更新cordova：

npm update -g cordova

cordova更新完成后，还需要更新项目：

cordova platform update android
