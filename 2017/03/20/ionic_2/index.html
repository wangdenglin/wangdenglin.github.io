ionic 常见问题2

1.ionic中如何打开微信（或者其他应用）
使用协议weixin://
<a class="button button-block button-light" href="weixin://">打开微信</a>
<access origin="weixin:*" launch-external="yes"/> //打开微信的

2.ionic.Platform.exitApp()退出后，再进入应用 splashscreen不显示
在config.xml文件里添加
<preference name="SplashShowOnlyFirstTime" value="false" />
3.微信的分享功能
http://www.iamsuperman.cn/2016/10/09/ionic/ionic-wechat-share/
4.隐藏后退按钮的文字
CSS方式
/隐藏后退按钮文字/
.bar .buttons .back-text { display: none !important; }
js方式:在config()中添加
$ionicConfigProvider.backButton.previousTitleText(false);
ionic在装载ion-view的时候如何隐藏ion-nav-header
只需为ion-view添加属性：hide-nav-bar=“true”即可实现隐藏父ion-nav-bar
pdf
http://www.iamsuperman.cn/2016/08/19/angular/angular-download-pdf/
