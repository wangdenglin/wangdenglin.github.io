nodejs path模块

规范化路径
path模块中的normalize()函数来规范路径字符串
var path=require('path'); path.normalize('/foo/bar//baz/asdf...') //=>'/foo/bar/baz/asdf.'
path.join()函数，可以连接多个路径字符串,并对路径规范话
例var path=require('path');
path.join('/foo','/bar','baz')
//=>/foo/bar/baz
path.dirname()获取文件路径的目录部分，例如：
path.dirname('/foo/bar/asdf/aaa.txt')
=>/foo/bar/asdf;
path.basename()获取文件路径下的文件名，例如：
path.basename('/foo/bar/baz/asdf/bbb.html')
=>bbb.html
另外可以将扩展名，传递给path.basename函数，作为其第二个参数，以此在文件中提取扩展名，例子如下：
path.basename('foo/bar/baz/ccc.html','.html')
=>ccc
path.extname()获取扩展名。
path.exists()确定路径是否存在(需使用回调函数如下)


path.existsSync()可直接返回路径是否存在。
