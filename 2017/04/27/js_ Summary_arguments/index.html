JS arguments 详解
在函数代码中，使用特殊对象 arguments，开发者无需明确指出参数名，就能访问它们。并且arguments
的长度是取决与实参而非形参。另外arguments是类数组，我们可以用Array.prototype.slice.apply(arguments)将其转化为真正的数组。(虽然即使不转化也能使用，但更严谨的方式还是转化比较好)

callee 属性是 arguments 对象的一个成员，仅当相关函数正在执行时才可用。callee 属性的初始值就是正被执行的 Function 对象，这允许匿名的递归函数。(print()打印)
